# CopterSonde app installation on Herelink using Windows computer

1. Enable developer mode on the Herelink. Steps for this are universal for every android device and can be found online.
2. Go to Developer options, enabled USB debbugging.
3. Connect the Herelink to the computer via USB cable
4. Swipe down the top panel and click on "USB charging this device". Switch use USB to "Transfer files"
5. A file explorer pops up showing the mass storage. Double click and go to Download folder.
6. Retrieve the apk file produced by Github under Actions or from a local compilation.
7. Drop the apk file in the Download folder of the Herelink
8. Go to Settings -> Storage -> Internal shared storage -> Explore -> Download
9. Click on the apk file and agree to install the app
10. Go to the app drawer, you will find the CopterSonde GCS app there.
11. Press and hold. Add to favorites. App will now show up on the Home page
12. Run the app. Accept permission to use storage.
13. Start using the CopterSonde GCS android app
